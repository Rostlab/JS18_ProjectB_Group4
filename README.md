## Requirements
* [node.js]
* [Express]
* [compromise](http://compromise.cool/)

### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ cd JS18_ProjectB_Group4
$ npm install
$ npm start
```
Visit http://localhost:3000 in the browser.
